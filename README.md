# Student Toolkit

A simple and useful web application for students that provides academic and utility tools in one place.

## Features

*  SGPA Calculator (with credits)
*  CGPA Calculator (semester-based)
*  Attendance Tracker
*  Password Generator

## 🌐 Live Website

 https://tanisha-prakash.github.io/student-toolkit/

##  Built With

* HTML
* CSS
* JavaScript

##  Purpose

This project was built to help students easily calculate their academic performance and access useful tools in one place.
